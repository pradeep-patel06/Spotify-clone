Spotify Clone 🎵

A modern Spotify-like music player built using HTML, CSS, and JavaScript. Play songs, manage favourites, view recently played tracks, shuffle, repeat, control volume, and toggle light/dark mode.

🚀 Live Demo: https://spotify-clone-ww2z.vercel.app

Features
Play / Pause music
Next / Previous song
Shuffle & Repeat
Song progress bar with clickable seeking
Volume control & Mute
Search songs dynamically
Add up to 3 favourites
Recently played songs list
Create playlist (alert only)
Toggle Light / Dark mode
Responsive, modern Spotify-like UI
Technologies Used
HTML5 – Structure of the app
CSS3 – Styling and responsive design
JavaScript (ES6+) – Player functionality, playlist, search, and controls

How to Use
Clone the repository or download the project.
Open index.html in any modern web browser.
Click on a song to play it.
Use the controls to Play / Pause, Next / Previous, Shuffle, Repeat, Volume, and Mute.
Search for songs using the search box.
Add songs to favourites (max 3).
View recently played songs.
Toggle Light/Dark mode using the 🌙 button.
Create a playlist (alerts only, no storage yet).
