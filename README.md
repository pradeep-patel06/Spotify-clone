Spotify Clone

A simple Spotify Clone built using HTML, CSS, and JavaScript.

##  Features
- Play / Pause music
- Next / Previous song
- Progress bar
- Responsive UI
- Modern Spotify-like design

##  Technologies Used
- HTML5
- CSS3
- JavaScript
